# Фотоархив компании

Учебный каркас Flask-приложения для корпоративного фотоархива. Сейчас проект настроен без авторизации, чтобы было проще тестировать страницы и макет.

## Что уже есть

- главная страница с последними фотографиями и счетчиком архива;
- общий список фотографий;
- детальная карточка фотографии;
- поиск по названию, автору и описанию;
- форма добавления записи;
- загрузка изображения в `static/uploads` или временный ввод имени файла вручную;
- SQLite-база `photo_archive.db`, которая создается автоматически при запуске.

## Структура

- `app.py` - Flask-приложение, маршруты и модель `Photo`;
- `templates/` - HTML-шаблоны страниц;
- `static/css/style.css` - стили интерфейса;
- `static/uploads/` - папка для загруженных изображений;
- `requirements.txt` - зависимости проекта.

## Запуск

```bash
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
python app.py
```

После запуска откройте:

```text
http://127.0.0.1:5000
```

## Что можно добавить дальше
