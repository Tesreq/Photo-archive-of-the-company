# Фотоархив компании

Простой каркас учебного приложения на Flask для хранения корпоративных фотографий.

## Что сделано

- `app.py` — точка входа, блюпринт не используется, но архитектура легко расширяется
- `templates/` — базовые странички: главная, архив, детальная запись, загрузка и поиск
- `static/css/style.css` — минимальный адаптивный стиль
- `requirements.txt` — зависимости для Flask + SQLAlchemy + миграций
- `photo_archive.db` создаётся автоматически при первом запуске

## Структура

- `app.py` — сервер + маршруты
- `templates/` — HTML-шаблоны
- `static/css/style.css` — внешний вид
- `requirements.txt` — зависимости

## Запуск

1. Создать виртуальное окружение

```bash
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
```

2. Запустить приложение

```bash
python app.py
```

3. Открыть в браузере

http://127.0.0.1:5000
